# funkyelite-archive

A preserved snapshot of **funkyelite.com** — my original online Guest List Portal
for London's premier clubs from the late '90s and 2000s.

Back then, funkyelite billed itself as *"London's Premier VIP Guestlist Club Site"* —
a place to get on the list for the city's most sought-after nights. This repository
keeps the original site exactly as it was: hand-built HTML, sliced Photoshop graphics,
guestlist and membership forms, venue pages, and all the artefacts of the early web.

## Why this archive exists

This is a time capsule. It's being kept for posterity ahead of a full **redesign and
revamp**, so the original can live on even as a new version takes shape.

## Then vs. Now

| | Then (late '90s–2000s) | Now (the revamp) |
|---|---|---|
| **Build** | Hand-coded HTML, table layouts, sliced Photoshop graphics | Modern, responsive redesign |
| **Guestlist** | HTML forms wired to the early web | A rebuilt, streamlined sign-up flow |
| **Look** | Classic 2000s club-site aesthetic | A fresh take on the FunkyElite identity |
| **Status** | Archived and frozen in time | Actively being reimagined |

The goal isn't to erase the original — it's to honour it. This archive is the "then";
the revamp is the "now."

## What's inside

- `index.html` — the original landing page
- `Welcome/` — entry and selection pages
- `Venues/` — the featured London clubs
- `Gallery/` — event and club photography
- `GuestlistForm/` — the original guestlist sign-up
- `NewMemberForm/` — membership registration
- `images/`, `misc graphics/` — the site's original artwork and sliced graphics

## Status

Archived and read-only — a record of the original site. A redesigned, modern version
is on the way.
